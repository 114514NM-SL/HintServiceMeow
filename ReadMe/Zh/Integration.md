# HintServiceMeow 插件融合
## Scripted Event
Script event指令:
    名称: "HSM_CommonHint";
    别名: "HSM_CH" 
    类别: Broadcast;
    参数:
    - Players (Required)
        一个玩家变量， 代表了所有要显示文字的玩家 
    - Text (Required) 
        一个一段文字(string)， 代表了要展示的文字。您可以使用[End-Line]标签来换行。最多可分4行文字。
    - Duration (Not Required)
        一个数字，代表了显示的时间
